# House Price Prediction

This project is a Machine Learning model built to predict house prices based on various features using **Linear Regression**.

## Project Description
The goal of this project is to analyze housing data and build a predictive model that can estimate house prices accurately.

##  Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook  

## Machine Learning Model
- Linear Regression Algorithm  
- Data Preprocessing  
- Feature Selection  
- Model Evaluation  

## Outcome
The model helps in predicting house prices based on input features and improves understanding of regression techniques in Machine Learning.
